### Ruby version

`ruby '3.1.0'` 

### System dependencies

`$ bundle`
`$ yarn`

### Database

`$ rake db:create`
`$ rake db:migrate`

### Test

frontend:
`$ yarn test`

backend:
`$ rspec`

### Run dev server (react served by rails)

`$ bin/dev`
